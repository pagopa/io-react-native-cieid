# io-react-native-cieid

A bridge to add CieID auth on IO

## Installation

```sh
npm install io-react-native-cieid
```

## Usage

```js
import { IoReactNativeCieidView } from "io-react-native-cieid";

// ...

<IoReactNativeCieidView color="tomato" />
```
